# 瓦尔登湖

亨利·戴维·梭罗在瓦尔登湖畔自建小屋，独居两年又两个月，用亲身实验追问：人究竟需要多少物质才能生活？

本书不是小说，没有连续的情节，而是一部反思性的散文作品。梭罗在四季轮换中观察湖、森林、鸟兽与村民，思考经济、孤独、阅读、自然与更高的规律，最终呼吁每个人转向内心，去过一种更清醒、更自由的生活。
