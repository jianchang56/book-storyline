# 《傲慢与偏见》故事梗概

乡绅班纳特家有五个待嫁的女儿。有钱的单身汉彬格莱先生租下邻近的尼日斐花园，班纳特太太立刻把全副心思放在女儿的婚事上。大女儿吉英与彬格莱一见钟情，二女儿伊丽莎白却因为偏见，对彬格莱的好友达西先生充满反感；达西先生虽出身显赫、年入一万镑，却也因傲慢无礼得罪了伊丽莎白。两人从误解、冲突、书信剖白，到丽迪雅私奔引发的家庭危机，最终看清彼此的真心与品格。
