# 《子夜》故事梗概

《子夜》是茅盾的代表作。故事发生在 1930 年 5 月到 7 月的上海，民族资本家吴荪甫雄心勃勃地想以益中信托公司整合华商实业，对抗以赵伯韬为代表的买办金融势力。然而公债市场的豪赌、工厂工人的罢工、家乡双桥镇的农民暴动，以及家庭成员的离散，把他一步步逼入绝境。小说以吴荪甫在公债市场的惨败收尾，写出了民族资本在内外夹缝中的艰难与悲剧。
