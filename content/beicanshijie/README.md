# 《悲惨世界》故事梗概

冉阿让因偷一块面包被判十九年苦役，出狱后受尽冷眼，直到迪涅主教以宽恕与银烛台唤醒他的良知。他化名马德兰兴办工业、救济穷人，成为市长，却为救无辜者再次暴露身份。他带着女工芳汀的女儿珂赛特逃亡，在巴黎隐居多年。青年马吕斯爱上珂赛特，而一八三二年六月的街垒起义，将所有人的命运推向高潮。
