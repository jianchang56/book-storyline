# 《西厢记》故事梗概

《西厢记》是元代王实甫创作的杂剧，写唐代贞元年间，书生张君瑞与崔相国之女莺莺在普救寺相遇、相恋，历经贼兵围寺、老夫人赖婚、红娘传书、长亭送别、状元及第、郑恒争婚等波折，最终有情人终成眷属的爱情传奇。全剧以“愿普天下有情人的都成了眷属”作结，是中国古典戏曲中歌颂自由爱情的经典之作。
