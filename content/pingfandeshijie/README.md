# 《平凡的世界》故事梗概

上世纪七十年代至八十年代，陕北双水村孙家在贫困与变革中艰难前行。老大孙少安早早辍学务农，撑起全家；老二孙少平在县高中靠黑高粱面馍度日，却借书籍打开精神天地。少安与村支书之女润叶相恋却因门第被迫分离，娶了贺秀莲；少平毕业后不甘困守农村，到黄原揽工、下大牙湾煤矿，与省报记者田晓霞相爱。

农村实行生产责任制后，少安的砖窑几经沉浮终于成功；少平在黑暗的矿井中挣得尊严。然而晓霞在抗洪采访中牺牲，少平也在矿难中毁容。他拒绝调往省城，选择回到矿山。润叶在丈夫车祸残疾后理解了责任，秀莲却在日子刚好时身患重病。
