# 《童年》故事梗概

阿列克塞三岁丧父后，随母亲投奔外祖父卡什林家。染坊主一家充斥着两个舅舅为分家而产生的仇恨、外祖父的暴戾与贪婪，以及底层俄国社会的贫困与粗鄙。在这片黑暗里，外祖母的慈爱、小伙计茨冈的仗义、房客“好事情”的沉思，以及街头伙伴们的真情，成为阿列克塞成长中稀少却珍贵的光亮。母亲再婚后的不幸、继父的家暴与小弟弟的夭折，最终把阿列克塞推向更为残酷的现实，也迫使他告别童年，独自“走向人间”。
