# 《德伯家的苔丝》故事梗概

英国威塞克斯乡村少女苔丝·德北菲尔德因家族虚荣心被送往特兰里奇，遭阿历克·德贝维尔诱骗失身；她在泰波塞斯奶牛场重获新生，与安琪儿·克莱尔相爱成婚，却在新婚之夜因坦白过去而被抛弃。苔丝在贫困与流言中挣扎求生，最终在阿历克再度纠缠、安琪儿迟来的悔悟之间，走向毁灭性的悲剧结局。
