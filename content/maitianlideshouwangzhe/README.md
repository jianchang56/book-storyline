# 《麦田里的守望者》故事梗概

十六岁的霍尔顿·考尔菲德因为功课不及格被潘西中学开除。他不敢马上回家，独自一人在纽约游荡了将近两昼夜。在这两天里，他遇见了虚伪的同学、势利的夜总会客人、敲诈勒索的电梯工，也遇见了让他重新相信纯真的妹妹菲芘。他唯一的梦想，是站在麦田边的悬崖上，把跑向危险的孩子一个个捉回来。
