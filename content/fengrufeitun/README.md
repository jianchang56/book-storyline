# 丰乳肥臀

作者：莫言

《丰乳肥臀》以母亲上官鲁氏的一生为轴，写她在1900年德国屠村中幸存，裹脚嫁入上官铁匠家，因丈夫不育而辗转借种，生下八女一男；又以九位子女的命运为网，铺展高密东北乡从日军入侵、国共内战、土改、大跃进、文革到改革开放近百年的动荡。母亲以乳房和坚韧养活一家人，而儿子上官金童终生困于恋乳癖，在时代浪潮中懦弱漂流。莫言以丰沛的感官描写与魔幻笔法，把个人苦难、家族兴衰与民族创伤熔铸成一部关于生命、土地与欲望的史诗。
