# 《双城记》故事梗概

法国医生曼内特在巴士底狱被秘密囚禁十八年后重获自由，女儿露西跨越海峡把他接回伦敦。十八年的冤屈没有磨灭父女之间的亲情，却像阴影一样笼罩着曼内特医生的记忆。与此同时，一位放弃法国贵族身份、隐姓埋名在伦敦生活的青年查尔斯·达尔内，因被控通敌而站上老贝勒的被告席。他与曼内特一家相遇、相知，最终与露西结为夫妻。然而巴黎的怒火正在积蓄，被压迫的人民拿起武器，旧日的血债与新仇交织，达尔内被迫返回故乡，陷入革命的审判与复仇之中。最终，一个与他相貌酷似的堕落律师西德尼·卡尔顿，以生命换取了他和露西一家的未来。
