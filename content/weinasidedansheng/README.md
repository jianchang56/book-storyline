# 维纳斯的诞生

文艺复兴时期的佛罗伦萨，布商之女亚历山德拉·塞奇渴望成为画家，却被时代的枷锁困在家中。神秘北方画家的到来点燃了她的艺术梦想，也把她卷入一场关于欲望、信仰与身份的漫长挣扎。

本书讲述亚历山德拉如何在“豪华者洛伦佐”去世、法军入侵、萨伏那罗拉宗教狂热与瘟疫的阴影中，被迫进入一桩交易的婚姻，最终以修女路克丽西娅的身份度过余生，并把一生的秘密刺在身体上的故事。
