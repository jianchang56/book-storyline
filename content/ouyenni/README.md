# 《欧也妮·葛朗台》故事梗概

索缪城的老箍桶匠葛朗台是远近闻名的吝啬鬼，他积攒下巨额家财，却让妻女过着黯淡拮据的生活。女儿欧也妮天真沉静，在二十三岁生日那天遇见了从巴黎来的堂弟夏尔，第一次尝到心动滋味。然而夏尔父亲破产自杀的消息接踵而至，欧也妮在堂弟的悲痛中献出全部积蓄与真情，却不知这段爱情终将被金钱与门第碾碎。
