# 《钢铁是怎样炼成的》故事梗概

保尔·柯察金从车站食堂的童工成长为百折不挠的无产阶级战士。他的故事横跨国内战争、战后重建和党内斗争，以钢铁般的意志对抗命运的一次次打击，最终在瘫痪失明后通过文学创作重返战斗行列。
