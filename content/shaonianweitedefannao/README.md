# 少年维特的烦恼

《少年维特的烦恼》（*Die Leiden des jungen Werthers*）是约翰·沃尔夫冈·冯·歌德于 1774 年发表的书信体小说，德国“狂飙突进”运动的重要成果。

本书以青年维特写给挚友威廉的书信为主体，讲述他离开城市、寄居乡间瓦尔海姆，爱上已订婚的绿蒂，又在绿蒂的丈夫阿尔贝特归来后陷入无望情感，最终走向毁灭的悲剧。
