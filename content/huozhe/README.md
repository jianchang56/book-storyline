# 活着

作者：余华

地主少爷徐福贵赌光百亩家业，气死父亲，被国民党拉壮丁，又在战场上捡回一条命。回家后他分到了五亩地，以为日子能重新来过，没想到时代一波接一波的动荡，把母亲、妻子家珍、儿女有庆与凤霞、女婿二喜、外孙苦根，一个一个从他身边带走。最后只剩下一头老牛，还有老人自己。

《活着》不渲染苦难，只让一个人平静地讲述自己如何失去一切，又如何继续活着。
