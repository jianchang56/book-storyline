# 史记

本目录收录《史记》一百三十卷的连续阅读梗概，由 Storyline 项目整理生成。

## 关于本书

《史记》为西汉司马迁所撰，是中国第一部纪传体通史。全书上起黄帝，下迄汉武帝，分本纪、表、书、世家、列传五体，凡一百三十篇，约五十二万六千字。太史公以“究天人之际，通古今之变，成一家之言”自任，记事兼及政治、军事、制度、文化与各色人物，被鲁迅誉为“史家之绝唱，无韵之《离骚》”。
